Privacy

NanoGPT does not store your conversations. The assistant only sends page content to NanoGPT when you attach context or submit a prompt.

  • Local chat history, settings, and any saved API key stay in this browser extension storage.
  • Attached pages, elements, images, and marked areas are included in the prompt you send.
  • Web search is only used for a prompt when the Web search toggle is on.
  • Webpages cannot read your extension chat history or saved API key.
  • The browser assistant source code is public at github.com/Nano-GPT-com/nanogpt-browser-assistant.
