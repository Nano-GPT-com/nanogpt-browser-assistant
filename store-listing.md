## Chrome Web Store

Summary:
Use NanoGPT on webpages to summarize, explain, translate, and chat with page context.

Description:
NanoGPT Browser Assistant brings NanoGPT into your browser side panel so you can work with the page you are already viewing.

Attach an entire page, a selected page element, an image, or a marked area. Then summarize it, explain it, translate it, or ask a follow-up question with the attached context. When the page alone is not enough, turn on web search from the composer.

Features:
- Side panel chat powered by NanoGPT.
- Entire page, element, image, and area attachments.
- Summarise, Explain, and Translate modes.
- Optional web search toggle.
- Model selection inside the assistant.
- Floating page button and keyboard shortcut.
- Local conversation history inside the extension.
- Sign in with NanoGPT or save a NanoGPT API key locally.

Privacy and permissions:
NanoGPT does not store your conversations. The extension needs access to webpages so it can read the page, selected elements, images, or marked areas when you ask it to. It uses storage to save local settings, local chat history, and an optional API key in your browser. The extension sends content to NanoGPT only when you submit a prompt or explicitly attach page context. The browser assistant source code is public at https://github.com/Nano-GPT-com/nanogpt-browser-assistant.

## Firefox Add-ons

Summary:
Use NanoGPT on webpages to summarize, explain, translate, and chat with page context.

Description:
NanoGPT Browser Assistant adds a NanoGPT sidebar to Firefox.

Use it to attach the page you are reading, a specific element, an image, or a marked area. Ask NanoGPT to summarize, explain, translate, or answer questions using that context. You can also choose models and turn web search on or off for a prompt.

Features:
- Sidebar chat powered by NanoGPT.
- Entire page, element, image, and area attachments.
- Summarise, Explain, and Translate modes.
- Optional web search toggle.
- Model selection inside the assistant.
- Floating page button and keyboard shortcut.
- Local conversation history inside the add-on.
- Sign in with NanoGPT or save a NanoGPT API key locally.

Permission notes for review:
NanoGPT does not store user conversations. The add-on requests webpage access so it can read page text, selected elements, images, or marked areas only when the user asks it to. Storage is used for local settings, local chat history, and an optional locally stored API key. Context is sent to NanoGPT only after the user submits a prompt or explicitly attaches page context. The browser assistant source code is public at https://github.com/Nano-GPT-com/nanogpt-browser-assistant.
